Tintify is a Chrome extension that adds a customizable overlay to webpages to reduce glare, improve readability, and minimize eye strain. The extension includes a responsive settings panel where users can adjust overlay color, opacity, and intensity, with their preferences automatically synced across devices.

Uses React and TypeScript to build a fast, responsive, and customizable UI.
Uses AWS DynamoDB to store user overlay preferences so settings persist across devices.
Uses AWS API Gateway to connect the extension to backend services for reading and updating user data.
